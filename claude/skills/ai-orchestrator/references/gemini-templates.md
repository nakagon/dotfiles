# Gemini 依頼テンプレート集

AI Orchestrator で Gemini に依頼する際の具体的なテンプレート。
目的に応じて適切なテンプレートを選択し、コンテキストを埋めて使う。

## コードレビュー

```
mcp__gemini-cli__geminiChat({
  prompt: `以下の [言語] コードをレビューしてください。

## 確認観点
- バグ・論理エラー
- パフォーマンスの問題
- セキュリティ上の懸念
- 可読性・保守性

## コンテキスト
- プロジェクト: [概要]
- 技術スタック: [Python/FastAPI, TypeScript/Next.js 等]
- このコードの役割: [何をするコードか]

## コード
\`\`\`[lang]
[コード]
\`\`\`

## 出力形式
指摘は以下の重要度で分類してください:
- **Critical**: 即時修正が必要
- **Warning**: 修正を推奨
- **Info**: 改善提案

各指摘に具体的な修正案を添えてください。`,
  model: "gemini-2.5-pro"
})
```

## 技術リサーチ

```
mcp__gemini-cli__geminiChat({
  prompt: `[トピック] について、以下の観点で調査してください:

1. 現在の主流なアプローチ
2. 主要ライブラリ/ツールの比較（メンテナンス状況、パフォーマンス、コミュニティ）
3. 以下の制約条件との親和性:
   - [制約1]
   - [制約2]
4. 潜在的なリスクや注意点

可能であれば具体的なコード例も含めてください。
結論として、推奨アプローチを1つ選び理由を説明してください。`,
  model: "gemini-2.5-pro"
})
```

## ウェブ検索

```
# 一般的な技術検索
mcp__gemini-cli__googleSearch({
  query: "[具体的な技術キーワード] [年] best practices",
  raw: true
})

# セキュリティ脆弱性検索
mcp__gemini-cli__googleSearch({
  query: "[ライブラリ名] CVE vulnerability security advisory",
  raw: true
})

# エラー調査
mcp__gemini-cli__googleSearch({
  query: "[正確なエラーメッセージ] [フレームワーク名] fix",
  raw: true
})
```

## セキュリティ分析

```
mcp__gemini-cli__geminiChat({
  prompt: `以下のコードについてセキュリティ分析をしてください:

## 対象
[認証フロー / APIエンドポイント / データ処理 等]

## コード
\`\`\`[lang]
[コード]
\`\`\`

## 確認観点
- OWASP Top 10 の該当項目
- 認証/認可の問題
- 入力検証の不足
- インジェクション攻撃のリスク
- 機密データの取り扱い

各脆弱性について:
1. リスクレベル（Critical/High/Medium/Low）
2. 攻撃シナリオ
3. 具体的な修正案
を提示してください。`,
  model: "gemini-2.5-pro"
})
```

## 使い分けガイド

| 目的 | ツール | モデル |
|------|--------|-------|
| 深い分析・レビュー | `geminiChat` | `gemini-2.5-pro` |
| 素早い確認・質問 | `geminiChat` | `gemini-2.5-flash` |
| 最新情報の調査 | `googleSearch` | (自動) |
| 既知の問題検索 | `googleSearch` → `geminiChat` | 検索後に深堀り |
